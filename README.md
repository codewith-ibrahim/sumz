# 🚀 React + Vite 

This project is a minimal setup using **React** with **Vite** including hot module replacement (HMR), ESLint rules, and a smooth development experience.

## 📦 Features

- ⚛️ React
- ⚡ Vite (or Next.js)
- 💅 ESLint & Prettier
- 🔥 HMR (Hot Module Replacement)
- 🎨 Tailwind CSS (if you're using it)
